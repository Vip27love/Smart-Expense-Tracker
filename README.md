# 💰 Smart Expense Tracker

A Python-based Expense Tracker application that helps users manage their income, expenses, and monthly budget efficiently.

---

# 📌 Features

- User Registration
- User Login
- Set Monthly Budget
- Add Income
- View Income
- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- Monthly Expense Report
- Financial Summary
- Budget Alert
- Pie Chart Analysis
- Bar Chart Analysis
- Export Expenses to CSV

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Vip27love/Smart-Expense-Tracker.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Database

Open **config.py**

```python
DB_CONFIG = {
    "host": "localhost",
    "user": "root",
    "password": "YOUR_PASSWORD",
    "database": "expense_tracker"
}
```

### Run Project

```bash
python main.py
```
